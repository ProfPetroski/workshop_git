git init irá iniciar a estrutura de versionamento no diretório atual ```bash git init meu_projeto ``` - git init meu_projeto irá criar um diretório de nome "meu_projeto" e criar a estrutura dentro deste diretório. --- class: top # Monitorando mudanças - Para que um arquivo seja rastreado, devemos executar o seguinte comando: ```bash git add carros.txt #ou git add . # o ponto (.) inclui todos os arquivos não rastreados ou modificados ``` .image-55[![Imagem](/assets/img/slides/workshop_git/git_add.png)] --- class: top # Gravando um arquivo no repositório (commit) - Para visualizar o histórico de commits ```bash git log ``` - Para mostrar as alterações de um commit ```bash git show ``` - Commit deve ser especificado pela chave .image-50[![Imagem](/assets/img/slides/workshop_git/git_log.png)] --- class: top # Ramificações (branching) - Branch é uma ramificação do repositório - Alterações (commits) ocorrem na branch - Muito útil para trabalhos colaborativos - Branchs de desenvolvimento facilitam o controle Para listar as branches do nosso repositório, devemos executar o comando: ```bash git branch ``` .image-50[![Imagem](/assets/img/slides/workshop_git/git_branch.png)] --- class: top # Listar e criar branch - Como listar todas as branches? ```bash git branch -v ``` - Como criar uma branch? ```bash git branch nome_da_branch ``` - Modo mais rápido de criar e deixá-la como corrente: ```bash git checkout -b nome_da_branch ``` --- class: top # Alterar branch corrente e apagar - Como mudar de branch? ```bash git checkout nome_da_branch ``` - Apagar uma branch em seu repositório local ```bash git branch -d nome_da_branch ``` - Apagar uma branch remota ```bash git push origin --delete nome_da_branch ``` --- class: top # Mesclar branch -
